A tesseract ocr

A-1 tesseract ocr简介

一句话概括，tesseract ocr（下文简称为tess）是惠普研发的一个文字图像识别组件，原本是用来给识别扫描仪扫出来的图片，后来进化得越来越妖，连一些简单的扭曲验证码图片也能识别成文字了。
比较正规的介绍请看这里↓
tess官网
